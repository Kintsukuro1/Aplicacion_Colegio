# Resumen — Características

- Propósito: resumen corto de la plataforma y sus módulos principales.
- Stack: Django + DRF, React 18 + Vite, templates Django + JS vanilla, Chart.js y CSS modular.
- Features: dashboard, calendario, reuniones, comunicados, mensajería, notificaciones, biblioteca, finanzas, import/export y onboarding.
- Hecho a mano vs framework: varias pantallas legacy están hechas con templates y JavaScript propio; las nuevas usan React y componentes reutilizables.
- Dónde revisar: services del backend, API layer, templates legacy y frontend React.
- Clave de auditoría: distinguir si la evidencia viene de services, signals o auditoría.
